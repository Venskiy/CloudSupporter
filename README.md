# CloudSupporter

### Prerequisites
* Gstreamer-1.0
* Python 3.6


### How to Build

1. Clone this repository or download a ZIP file of it:
    * `https://github.com/Venskiy/CloudSupporter.git`
    * `cd CloudSupporter`
2. Install dependencies:
    * `pip install -r requirements.txt`
3. Run the app:
    * `python app.py /dropbox/[path to file on dropbox]`
